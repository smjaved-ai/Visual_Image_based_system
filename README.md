# Visual_Image_based_system
**********************************************************
Visual Product Similarity & Image-Based Recommendation System 
**********************************************************

###Description###
The objective of this project is to build a image retrieval and similarty search system built using the Cars dataset from Hugging Face:
●	Extract high-quality visual embeddings from product images using deep learning

●	Build a fast similarity search engine to retrieve visually similar products

●	Rank and return Top-K similar products for any input image

●	Demonstrate a scalable, production-like recommendation pipeline

**********************************************************

###Pre-Requisites####
  Latest google chrome or Microsoft Edge Browser
  Python (3.8 or later recommended)
  Streamlit
  SQL database (MySQL Workbench)

**********************************************************
